# Fuckftp
使用C++写的轰炸ftp程序，引用了winscp用作连接ftp并上传文件，Releases里打包了winscp，运行install.bat会复制fuckftp以及依赖文件至C:/Temp
# 运行命令
C://Temp//fuckftp.exe
